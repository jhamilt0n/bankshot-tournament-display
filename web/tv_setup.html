üîß Troubleshooting Tips

Can't connect to .local address?
Use the IP address version instead (shown above). Some older TVs don't support mDNS.

Display not switching?
Refresh the page on your TV browser. The system checks for updates every 30 seconds.

Need help?
Add ?debug=1 to the end of the URL to see status messages in the corner.
